## Config overrides
```json
{

}
```

## Hypothesis
Disabling mmap forces eager load of all weights into RAM, eliminating page-fault overhead during expert access at the cost of higher RSS.

## Rationale
Separates 'trust OS page cache' cost from 'no page faults' benefit.
