# Experiment 001 — torch.no_grad() context

- **Category:** runtime
- **Applied on:** baseline config
- **Data source:** MEASURED (`measured=true`)
- **Decision:** **DISCARD**

## Hypothesis
Disabling autograd bookkeeping reduces per-step overhead.

## Rationale (bottleneck targeted)
Generation never needs gradients; baseline leaves autograd active.

## Prior expectation
Small but free win on most backends.
