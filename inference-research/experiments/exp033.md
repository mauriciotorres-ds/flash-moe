# Experiment 033 — fp16 KV cache, fp32 compute

- **Category:** memory
- **Applied on:** best config
- **Data source:** MEASURED (`measured=true`)

## Hypothesis
Store K/V in fp16 to halve cache memory while computing in fp32.

## Rationale (bottleneck targeted)
KV-cache memory bottleneck at long context on 24GB.

## Prior expectation
Memory win; small/no speed change for short prompts.
