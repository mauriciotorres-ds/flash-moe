# Experiment 012 — Offloaded KV cache

- **Category:** memory
- **Applied on:** best config
- **Data source:** MEASURED (`measured=true`)
- **Decision:** **DISCARD**

## Hypothesis
Offloading cache to CPU frees GPU memory for weights.

## Rationale (bottleneck targeted)
Targets the 24GB memory ceiling for long contexts.

## Prior expectation
Likely slower for short prompts; helps long-context only.
