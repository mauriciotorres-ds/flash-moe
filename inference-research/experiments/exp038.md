# Experiment 038 — Cached GenerationConfig reuse

- **Category:** runtime
- **Applied on:** best config
- **Data source:** MEASURED (`measured=true`)
- **Decision:** **KEEP**

## Hypothesis
Reuse a prebuilt GenerationConfig to skip per-call validation.

## Rationale (bottleneck targeted)
Per-call Python overhead in generate().

## Prior expectation
Tiny win; cheap to keep if non-negative.
