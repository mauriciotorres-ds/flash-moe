# Experiment 011 — Static KV cache

- **Category:** memory
- **Applied on:** best config
- **Data source:** MEASURED (`measured=true`)
- **Decision:** **KEEP**

## Hypothesis
Pre-allocated static cache avoids per-step reallocation and is required for clean torch.compile graphs.

## Rationale (bottleneck targeted)
Removes allocation churn; enables later compile combo.

## Prior expectation
Small win alone; unlocks exp29 combo.
